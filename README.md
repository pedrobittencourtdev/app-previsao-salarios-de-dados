# 💰 Previsão de Salários para Profissionais de Dados

Este projeto utiliza Machine Learning para prever o salário anual de profissionais com base em características como idade, gênero, nível de escolaridade, cargo e anos de experiência.

🔗 **[Acesse o App aqui!](https://tinyurl.com/data-salary-2026)**

## 🚀 Tecnologias Utilizadas
* **Python** (Linguagem principal)
* **Scikit-learn** (Criação do modelo de Machine Learning)
* **Random Forest** (Algoritmo de regressão utilizado)
* **MLflow** (Gestão de experimentos e métricas)
* **Streamlit** (Interface do usuário e Deploy)
* **Feature-engine** (Tratamento de dados e encoding)

## 📊 O Projeto
O modelo foi treinado com um conjunto de dados contendo informações salariais globais. Durante o desenvolvimento, utilizei o **MLflow** para monitorar a performance do modelo e garantir os melhores hiperparâmetros. 

O app final permite que qualquer pessoa insira seus dados e receba uma estimativa salarial em tempo real.
